📌 PyPhysics

“This project was developed with the assistance of Artificial Intelligence (AI) to support the design and implementation of interactive physics simulations.”

The main goal of PyPhysics is to support educators in making physics more engaging and visually accessible through interactive simulations. It is designed to help both students and teachers in Brazil and internationally, making physics concepts easier to understand through visualization and interaction.

⚙️ How to Run the Code

This project was developed and tested using Google Colab.

To ensure proper functionality and visualization, the code is organized into cells that should be executed in order.
### 1. First step: Import cell

Before running any simulation, execute the first cell, which contains all required imports and visualization settings:

import numpy as np
import matplotlib.pyplot as plt
import ipywidgets as widgets

plt.style.use('seaborn-v0_8-darkgrid')

### 2. Then run the remaining cells

After running the import cell, proceed to execute the other cells that contain the simulation logic and interface.

This order is important to ensure that all dependencies are loaded correctly and that the simulation runs as expected.
